# MMock Console UI

A modern web console for MMock HTTP mock server built with React and TypeScript.

## Features
- Real-time request monitoring
- Mock definition management
- Test scenario control
- Interactive API dashboard

## Development

### Prerequisites
- Node.js & npm ([install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating))

### Getting Started

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Technologies
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Build Output
The production build outputs to `../internal/console/ui` as configured in `vite.config.ts`.
